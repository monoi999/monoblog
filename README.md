📘 AI 기반 데이터 처리 및 분석 실무 과정

    - 정적인 그래프를 넘어선 시각화: Matplotlib을 넘어 Plotly와 Seaborn을 이용한 인터랙티브 차트 생성

    - 단순 암기가 아닌, 전자공학의 수학적 배경(선형대수, 통계)을 AI 모델링에 연결합니다.
    - 회귀와 분류: 센서 데이터를 이용한 수치 예측 및 이상치 탐지

    - 실습: 웨이퍼 결함 유형(Donut, Scratch 등) 자동 분류 모델 구현
- 3부: AI 웹 대시보드 제작 (결과물 구현)
    - 분석 결과를 단순한 '보고서'가 아닌 '동작하는 서비스'로 만드는 단계입니다
    - Streamlit & Dash 기초: HTML/CSS 없이 파이썬만으로 인터랙티브 웹 앱 만들기
    - UI/UX 디자인: 사용자 친화적인 필터, 사이드바, 실시간 업데이트 기능 구현
    - API 연동: 모델을 웹 서비스와 연결하여 실시간 추론 결과 보여주기
    - 실습: "나만의 반도체 공정 모니터링 대시보드" 제작
- 4부: 프로젝트 고도화 및 포트폴리오 (심화)
- 취업 시장에서 차별화될 수 있도록 '자동화'와 '배포' 개념을 추가합니다
- 워크플로우 자동화: make 툴이나 스케줄러를 활용한 데이터 파이프라인 자동화
- 금융 데이터 응용: 주식/ETF 데이터를 수집하여 자신만의 투자 지표 대시보드 만들기
