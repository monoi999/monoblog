AI 활용1 : 타이타닉 생존 인사이트

강의 개요 : 실전 데이터셋(타이타닉)으로 AI를 이용하여 전처리와 시각화하고 Streamlit으로 배포까지 만들기

❇️ 타이타닉 생존 인사이트 대시보드 미리보기

  1. 타이타닉 대시보드 : 대시보드 둘러보기

❇️ 데이터 처리·분석을 위한 절차

  1. 문제 정의(목표 설정): 무엇을 알고 싶은지 문제를 정의하고 필요한 가설을 세움
  2. 데이터 수집: 설문조사, DB 추출, 웹 크롤링 등을 통해 필요한 자료를 모음
  3. 데이터 탐색 (EDA): 기술 통계와 시각화를 통해 데이터의 특징과 패턴을 파악
  4. 데이터 전처리/정제: 중복 제거, 결측치 보정, 오타 수정, 변환 등 분석 가능한 형태(Clean Data)로 만드는 가장 중요한 단계
  5. 데이터 모델링 및 평가: 통계 기법이나 머신러닝 알고리즘을 적용해 결과를 예측하거나 분류하여 인사이트 도출
  6. 결과 해석 및 보고: 분석 결과를 시각화하여 통찰을 도출하고 의사결정에 활용

❇️ 타이타닉 데이터셋을 이용하여 단계별 진행 사항

  1. 문제 정의(목표 설정)

    타이타닉 데이터셋을 활용하여 생존율에 영향을 미치는 주요 요인을 분석하고, 이를 바탕으로 생존 예측 분석을 대시보드까지 구현하는 것을 목표로 한다.

    • 생존율에 영향을 미치는 주요 요인 분석: 객실 등급, 성별, 나이, 가족 수 등이 생존율과 어떤 관계가 있는지 탐색한다.
    • 생존 예측 모델링: 머신러닝 알고리즘을 적용하여 승객의 생존 여부를 예측하는 모델을 구축한다.
    • 결과를 시각화 및 인사이트 도출: 모델이 생존을 예측할 때 어떤 요인이 중요한지 확인하고 시각화한다.
    • 대시보드 구현: 분석 결과를 시각화하여 비전문가도 쉽게 이해할 수 있는 대시보드를 Streamlit으로 구현한다.
  2. 데이터 수집 및 로드

    Kaggle Titanic이나 파이썬의 seaborn 라이브러리를 통해 데이터를 가져온다. 또는 실습자료 "data/titanic.csv"에서 데이터를 로드할 수 있다.

  3. 데이터 파악 및 탐색적 데이터 분석 (EDA)

    • 데이터의 구성과 특징을 파악하기 위해 컬럼별 데이터 타입, 결측치 현황, 기초 통계량 등을 확인한다.
    • 주요 변수인 성별, 객실 등급, 나이, 가족수, 요금(fare) 등이 생존율과 어떤 관계가 있는지 확인하고 탐색 결과를 요약한다.

  4. 데이터 전처리(Data Cleaning & Engineering)

    • 결측치 처리: 나이(Age)는 객실 등급(Pclass)과 성별(Sex)별 중앙값으로 채우고, 승선 항구(Embarked)는 최빈값으로 채운다.
    • 컬럼 삭제: 객실 번호(Cabin)는 결측치가 너무 많아 삭제하고, 고유 식별자 역할을 하는 PassengerId와 Ticket도 분석에 필요 없으므로 삭제한다.
    • 새로운 컬럼 추가: Name 컬럼에서 호칭(Mr, Mrs, Miss 등)을 추출하여 Title 컬럼을 추가하고, 가족 수를 나타내는 Family 컬럼을 SibSp와 Parch를 합쳐서 추가한다.
    • 범주형 데이터 변환 (Encoding): 머신러닝 모델은 문자열을 이해하지 못하여 성별(Sex)을 숫자(male: 0, female: 1)로 변환한다.

  5. 모델링 및 평가

    생존 여부를 예측하기 위한 머신러닝 알고리즘을 적용하고, 모델의 성능을 평가한다.

    • 의사결정 나무(Decision Tree)를 사용하여 생존 여부를 예측하는 모델을 학습시킨다.
    • 모델이 생존 예측에 가장 중요하게 작용한 변수를 확인하여 도출한 인사이트를 정리한다.

  6. 결과 요약 및 대시보드 기획 (Streamlit 연계)

    분석 결과를 비전문가도 이해할 수 있게 정리하고 Streamlit을 이용하여 대시보드를 구현한다.

    1. 대시보드 제목 : 타이타닉 생존 인사이트
    2. 대시보드 상단 (Key Metrics)
      • 전체 생존율: 전체 승객 중 생존자의 비율 (게이지 차트)
      • 모델 정확도: 학습된 모델의 예측 성공률 (숫자 위젯, 예: 81%)
      • 주요 변수 Top 5: 생존에 가장 큰 영향을 준 변수 리스트
    3. 대시보드 본문 (Interactive Visualization)
      • 생존율 시뮬레이터: 사용자가 객실 등급과 성별을 선택하면 해당 조건에 따른 생존 확률을 실시간으로 계산하여 게이지 차트로 시각화
      • 변수별 생존율 분석: 성별, 객실 등급, 나이 그룹별 생존율을 막대 그래프로 시각화하여 비교
    4. 대시보드 하단 (Insights & Summary)
      • 생존율에 영향을 미치는 주요 요인과 모델의 예측 성능을 간결하게 정리하여 보고서 형태로 제공
      • 생존율과 관련된 주요 변수들의 시각화를 포함하여 사용자가 쉽게 이해할 수 있도록 한다.
      • 사용자가 선택한 조건에 따라 생존 확률이 어떻게 변화하는지 시각적으로 보여준다.
