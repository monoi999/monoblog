Todo 앱 개발

목표: SQLite3로 데이터를 저장하고 Tkinter GUI로 관리하는 데스크톱 앱 구현

1. 주요 스택

  • 데이터베이스: SQLite3
  • GUI: Tkinter
  • 배포: PyInstaller로 exe 파일 생성

2. 주요 기능

  • 할 일 추가
  • 전체 목록 조회
  • 완료 체크(update)
  • 삭제
  • 카테고리별 검색
  • 완료 상태별 검색

3. 추가 기능

  • 카테고리별 할 일 추가
  • 카테고리 예시: 업무, 가족, 개인, 쇼핑

4. 설계 아이디어

  • 테이블 컬럼: id, title, category, is_done, created_at
  • 화면 구성: 입력 영역, 필터 영역, 목록 영역, 실행 버튼 영역
  • 목록 선택 후 완료/삭제 작업 수행
  • 필터 변경 시 목록을 다시 조회
